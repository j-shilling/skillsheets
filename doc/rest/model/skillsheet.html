REST Model - SkillSheet

SkillSheets are sent between the client and server as JSON objects. The server will activate and restrict fields depending on the account type of the user making the request. (e.g. Certain fields will always be null when requested by a student.)

Fields

Always Visible

uuid

A string uniquely identifying this object. The UUID can never be null.

name

A string representing the display name

dueDate

A string representing the due date of the skill sheet. This will be null if the owner has not set this value.

Visible only to Teacher Accounts

These fields will all be null if requested by a student account.

visible

A boolean, true if the skillsheet is visible to students.

availableFrom

A string representing the date and time the skillsheet will become visible to students automatically. This will be null if the value has not been set by the user.

availableUntil

A string representing the date and time the skillsheet will become invisible to students automatically. This will be null if the value has not been set by the user.

averageGrade

A number representing the average student grade on this skillsheet.

Visible only to Student Accounts

These fields will all be null if requested by a teacher account.

grade

A number representing the grade of the requesting student on this skillsheet.
